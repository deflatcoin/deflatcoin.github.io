Buying and Selling DEFLAT

When negotiating make sure you are operating with token in the address:

0xe1E0DB951844E7fb727574D7dACa68d1C5D1525b

This is important because different ethereum tokens can have the same name.

Do not send tokens or ethereum to this address, use it for your watch list. Sending must be from one wallet to another.

You can buy and sell at the following Exchanges:

 

ForkDelta Exchange (Link to Pair)

LedgerDEX (Connect in LedgerDEX and Add DEFLAT Coin to watch list)

Ethermium Exchange (Link to Pair)

 

Make sure that you send Ether from a wallet that supports ERC20 tokens, otherwise you will not be able to interact with the DEFT tokens received.
